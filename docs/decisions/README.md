## 命名规范

- 文件名: `NNNN-中文标题.md` (中文用汉字不用拼音 — 与 user 偏好一致)
- 编号从 0001 开始
- 状态: `Proposed` / `Accepted` / `Superseded by NNNN` / `Deprecated`

## 必填字段

- Context (问题背景)
- Decision (决策 + 理由)
- Consequences (收益 + 风险)
- 验证清单
